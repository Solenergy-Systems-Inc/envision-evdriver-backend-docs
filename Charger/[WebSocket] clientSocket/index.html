    ???/charger/client-socket

    Notes

    The DTOs below are the messages to be sent via WebSocket.

    ClientSocketRequestDTO

    Attributes:

    Name Type Description
    action ActionEnum
    body Union[RemoteStartRequestDTO, RemoteStopRequestDTO]

    ActionEnum

    Values
    Start
    Stop

    RemoteStartRequestDTO

    Attributes:

    Name Type Description
    site_id int
    charger_id str
    connector_id int
    duration float

    IN HOURS
    payment_intent_id Optional[str]

    USE THIS FOR PAYMONGO. OTHERWISE, MONEY WILL BE DEDUCTED TO WALLET
